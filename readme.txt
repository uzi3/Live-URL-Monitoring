This script will perform live monitoring of URLs name resolution using powershell. For URL which is getting resolved, the URL name will be displayed in green color. The URL which is not getting resolved, URL name will be displayed in red color. This will refresh every 10 seconds. You need to provide list of URLs in a text file.

NOTE: The powershell process running this script will hog memory over the time, hence you require to keep a track of available memory and processor periodically.
